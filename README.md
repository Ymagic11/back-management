# 简介

back-management是基于vue3,ts和element-plus的后台管理模板.

项目目的:

- 熟悉vue3+ts
- 学习 composition api 风格

## 在线演示
地址:http://47.108.28.179:81
## 目录结构


```
back-management
├─ .vscode            # vscode拓展
├─ src                # 源码
│  ├─ assets          # 全局样式 type
│  ├─ components      # 公共组件
│  ├─ mixin           # 组合式函数
│  ├─ router          # 路由
│  ├─ server          # 接口
│  ├─ stores          # 全局store管理
│  └─ views           # 页面
├─ README.md
├─ tsconfig.json      # ts 编译配置
└─ vue.config.ts      # vite 配置
```

## Project Setup

```shell
npm install
```

### Compile and Hot-Reload for Development

```shell
npm run dev
```

### Type-Check, Compile and Minify for Production

```shell
npm run build
```
